# Elite PR Review

Review the currently opened Pull Request.

Before reviewing:
1. Read PR title.
2. Read PR description.
3. Read linked Jira if available.
4. Understand expected behavior.
5. Use VS Code Pull Request Review.
6. Review ONLY changed code.
7. Expand only for context.

Return:
- Executive Summary
- Findings
- Missing Tests
- Positive Feedback
- Production Readiness Score
- Risk Level
- Merge Recommendation
